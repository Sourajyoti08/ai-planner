# ai-planner
AI daily planner using Flask and Gemini API.

A small project I built while learning programming.
The idea is simple:
Sometimes I have many things to do, but my thoughts are messy.
Instead of writing a structured list, I just type whatever comes to mind, and the AI converts it into a clear plan for the day.
Example:

Input:

study maths revise chemistry workout finish assignment..maybe read a bit and meet a friend in the evening...

Output:

1. Study maths
2. Finish assignment
3. Revise chemistry
4. Workout
